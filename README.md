## 📖 Introduction

This repository contains the complete set of core backend training modules and logical infrastructure completed during the **Generation Brasil Full Stack Java Bootcamp**. 


## 🎓 About Generation Brasil
**Generation** is a global non-profit organization that transforms lives by preparing, placing, and supporting people into life-changing careers that would otherwise be inaccessible. The Full Stack Java Bootcamp is an immersive, high-intensity program designed to develop market-ready skills in enterprise software engineering, programmatic logic, soft skills, and collaborative agile environments.
